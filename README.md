# manufacturing-erp-dotnet-core-crud
# 制造业销售订单 API (Demo)

##  项目简介
这是一个基于 **.NET Core** 构建的 RESTful API 演示项目，主要用于展示在制造业 ERP 场景下，对销售订单模块的标准化 CRUD 操作。

## 🛠️ 技术栈
- **框架**: .NET Core 10.0 (根据你的实际填写)
- **ORM**: Entity Framework Core 
- **数据库**: SQL Server 2012+
- **文档**: Scalar/ OpenAPI

## 核心功能
- 销售订单的增删改查 (CRUD)

## 如何运行
1. 克隆本项目
2. 修改 `appsettings.json` 中的数据库连接字符串
3. 执行 `Update-Database` 
4. 运行项目，访问 `/Scalar` 查看接口文档
